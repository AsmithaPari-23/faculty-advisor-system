# Faculty Advisor System

A full-stack MERN application designed to streamline communication and academic management between faculty advisors and students. The system provides secure role-based access for managing student records, requests, and announcements through an intuitive web interface.

## Features

- Secure JWT-based authentication
- Role-based access for Faculty Advisors and Students
- Student profile management
- Request submission and approval workflow
- Announcement management
- Dashboard for academic information
- Responsive and user-friendly interface

## Tech Stack

**Frontend:** React.js, Tailwind CSS, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT

## Installation

```bash
git clone https://github.com/AsmithaPari-23/faculty-advisor-system.git
cd faculty-advisor-system
npm install
```

Configure the required environment variables and start both the frontend and backend servers.
